Welcome to KymFlow

This installer sets up the local KymFlow runtime, installs JupyterLab, and prepares a user workspace under ~/Documents/KymFlow.

First install: downloading and installing Python packages can take several minutes. The progress bar may sit still while that runs—this is normal.

What this installer will create
  • ~/Library/Application Support/kymflow-pkg for runtime files
  • ~/Documents/KymFlow/Examples for installer-managed examples
  • ~/Documents/KymFlow/Example-Data for installer-managed example data
  • ~/Documents/KymFlow/User for your own work
